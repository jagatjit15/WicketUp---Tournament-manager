# 🏏 WicketUp - Cricket Tournament Scorer

WicketUp is a **premium, all-in-one cricket tournament management platform** designed for professional tournament administrators and casual enthusiasts alike. It streamlines every aspect of a cricket tournament, from initial registration to live scoring and dynamic playoff generations.

---

## ✨ Key Features

### 🏆 Tournament Orchestrator
- **Dynamic Scheduling**: Effortlessly manage league matches with a 2D-drag-and-drop match grid.
- **Auto-Playoffs**: Support for both standard **Semi-Final/Final** and the **IPL-style Qualifier/Eliminator** systems.
- **Smart Reordering**: Reorder matches simply by renaming labels like "Match 5" in the grid manager.

### 📉 Live Points Table
- **Real-time Analytics**: Net Run Rate (NRR) and points update automatically as matches complete.
- **Mathematical Qualification**: Visual indicators (Green labels) show if a team has already qualified for the next stage.
- **Global Table**: Automatically manages the league standings based on wins, losses, and ties.

### 🔢 Pro-Level Scorer
- **Rule-Perfect Logic**: Accurate handling of legal balls, wides, no-balls, and byes.
- **Innings Transition**: Seamless shifts between first and second innings with automated "Winner" and "Tied" messaging.
- **Flexible Controls**: Setup overs and wickets per match individually.

### 📤 Universal Reports
- **Professional PDF**: Generate branded tournament summaries with structured data tables.
- **Social Sharing**: Export high-resolution dashboard images (PNG) perfectly sized for WhatsApp or social media.

## 🚀 Quick Start

WicketUp is a **no-server application**. You can run it directly from your computer:

1. **Clone the Repo**: 
   `git clone https://github.com/jagatjit15/WicketUp---Tournament-manager.git`
2. **Launch**: Simply open `index.html` in any modern web browser or serve it using Live Server (e.g., `python -m http.server 3000`).
3. **Usage**:
   - Create a tournament and add teams.
   - Go to the **Match Grid** to organize your schedule.
   - Visit the **Scorer** tab to start matches.
   - Use the **Dashboard** to see the standings and export your results.
